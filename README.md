# 🌦️ Weather Insights

A modern and responsive weather application built with **React.js** that provides real-time weather information, interactive weather charts, and a seamless user experience using live weather data.

---

## 📖 Overview

**Weather Insights** enables users to search for any city and instantly access accurate weather information. The application displays key weather metrics such as temperature, humidity, wind speed, and weather conditions. It also includes interactive charts for visualizing weather trends, making it easier to understand forecast data. The interface is clean, responsive, and optimized for both desktop and mobile devices.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time weather information
- 📊 Interactive weather charts
- 📈 Temperature trend visualization
- ☁️ Current weather conditions
- 💧 Humidity information
- 🌬️ Wind speed details

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Weather-Insights.git
```

### 2. Navigate to the project

```bash
cd Weather-Insights
```

### 3. Install dependencies

```bash
npm install
```

### 4. Create an environment file

Create a `.env` file in the project root and add your API key:

```env
REACT_APP_WEATHER_API_KEY=YOUR_API_KEY
```

### 5. Start the development server

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```

This project uses the **OpenWeather API** to fetch live weather data, including:

- Humidity

- Temperature Trends
